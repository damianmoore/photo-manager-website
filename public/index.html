Photonix

Photo Manager

This is a new web-based photo management application. Run it on your home server and it will let you find the right photo from your collection on any device. Smart filtering is made possible by object recognition, location awareness, color analysis and other algorithms.

Try live demo

We’re trying to build a community around Photonix. If you’d like to be notified when we release our 1.0 version among other things, please sign up to the mailing list. Thank you.
